Features
PARSER
KiCad Netlist Import
Load any .kicad_net file directly. Validates components, pins, and net consistency automatically.
PLACEMENT
Force-Directed + SA
Hybrid placement using force-directed simulation followed by simulated annealing refinement.
ROUTING
A* 3D Router
4-layer A* routing with via minimization, fallback strategies, and rip-up/reroute mechanism.
AI / ML
Routability Prediction
Gradient Boosting model predicts routing success in ~0.3ms, skipping poor placements before routing.
SCORING
Multi-Criteria Score
Completion rate, wire length, critical nets, compactness and via count — all combined into one score.
EXPORT
KiCad Export
Generates a .kicad_pcb file directly openable in KiCad PCB Editor. No manual conversion needed.
How to use
01
Download & Extract Click the Download button above. Extract the ZIP file anywhere on your PC — no installation needed.
02
Run SmartLayout.exe Open the extracted folder and double-click SmartLayout.exe. No Python required.
03
Load a Netlist Click "Load Netlist" and choose a .kicad_net file from the netlists folder included in the ZIP.
04
Configure & Run Pipeline Set board size, seed, and optimization mode. Click "Run Pipeline" and watch the PCB generate in real time.
05
Export to KiCad When done, click "Export" to save a .kicad_pcb file you can open directly in KiCad.
